This plugin enables and facilitate users to protect their intelligent properties through Copystrong script.

== Description ==

Copystrong is a middleware provider that protects intelligent properties. The IP owners can set up the pages to monitor and control visitor behaviors, such as copy & paste, right-click, print, drag and drop, or using proxy.

Copystrong services are triggered by a line of javascript, which users can inject into their own pages to the place where they want to protect the IP.

This plugin facilitate the protection process by automating the javascript injection.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/copystrong` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Acquire the protection code from Copystrong.
4. Inject the code into your pages. `[copystrong code='Your code']`

== Frequently Asked Questions ==

= How can I get my protection code? =

Please contact Copystrong for your code.

= How can Copystrong protect my Intelligent Properties? =

Copystrong put a listener to visitor behaviors, and record and control their actions.

= Do Copystrong install software on visitor's computer? =

No, Copystrong do not install anything on visitor's computer.

= Do Copystrong leave Cookie or Session on visitor's computer? =

No, all tracking and identifying is done annonymously, without leaving anything on visitor's computer.

= What if a visitor disable script? =

Copystrong will record that the script is disabled, with as much information as possible about such a visitor.

= Do I need Wordpress to use Copystrong? =

No. You may use any CMS or your own pages and inject the Copystrong script manually. This plugin facilitate use of Copystrong script at Wordpress.
